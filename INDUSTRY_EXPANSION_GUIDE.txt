# ── WHAT EACH PHASE DELIVERS ──────────────────────────────────────────────

# Phase 1 — Industry Profile Architecture
#   Config-driven sector selection, 3 new control catalogues
#   (NIST 800-53 full, HIPAA, PCI DSS + SOX ITGC), industry_service router

# Phase 2 — Healthcare Edition
#   PHI asset classification with risk multipliers, HIPAA breach
#   notification workflow with HHS/individual/media deadline tracking,
#   HIPAA Security Rule compliance tracker (all 3 safeguards)

# Phase 3 — Financial Edition
#   FAIR quantitative risk calculator (ALE in $), PCI DSS v4.0 full
#   12-requirement tracker, SOX ITGC 4-domain tracker, pre-built
#   financial and healthcare risk scenarios

# Phase 4 — Government Edition
#   NIST SP 800-53 Rev 5 full catalogue (8 families, 100+ controls),
#   FedRAMP/FISMA ATO lifecycle (categorization → active ATO),
#   POA&M creation and tracking with overdue detection

# Phase 5 — Multi-Tenant + Commercial Launch
#   Tenant provisioning with industry profile seeding, 30-day trial
#   management, tenant context injection via session state,
#   self-service onboarding page (loads as page 0)
